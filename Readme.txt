# Sistem Optimasi Distribusi Logistik

Aplikasi web sederhana berbasis PHP & MySQL untuk menentukan distribusi barang dengan biaya minimum.

## Fitur
- Input kapasitas gudang
- Input permintaan toko
- Perhitungan distribusi optimal
- Penyimpanan hasil ke database

## Teknologi
- PHP Native
- MySQL
- HTML & CSS

## Cara Menjalankan
1. Clone repository
2. Import `database.sql`
3. Jalankan XAMPP
4. Akses melalui browser

## Konsep
Aplikasi ini dibuat sebagai implementasi project Analisis dan Optimasi Sistem.
